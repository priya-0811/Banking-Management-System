# Banking Management System

Banking Management System is a software application developed to manage customer accounts and perform essential banking operations efficiently and securely.

## Features
- Customer account creation
- Deposit money
- Withdraw money
- Balance enquiry
- Fund transfer
- Transaction history
- User authentication
